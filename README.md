# Spellex
Hello!, *Spellex* is an extension to correct your spell.

# INSTALL INSTRUCTIONS

  First download the zip and do `npm install` on the terminal. then,

  Go to the URL `(your browser)://extensions`.  in the right upper corner, there will be a switch, 
  
  Saying `developer mode`, press it. after that, in the left upper corner will be a button that says,

  `Load unpacked extension` press it. Choose the dist of Spellex.

  Done!, Spellex will be added to your extensions so you can use it.

  (Note: you require a OpenRouter api key.)

  # Version

  Spellex's version right now is: `1.1`

  # End

  Hope you enjoy the extension!
